06 — Usage rules
Do
Use --accent only for brand marks, primary CTAs, and "currently running" states
Always pair the icon with the wordmark at 24px+ display sizes
Use JetBrains Mono for all on-chain data — addresses, hashes, amounts
Prefer dark canvas (#0A0A0B) as default background
Keep the shield-outline icon at ≥16px minimum — details disappear below that
Don't
Use accent green for severity signals — it conflicts with sev-clean
Add drop shadows, glows, or blur effects to the logo
Use the logo on colored backgrounds other than the four approved surfaces
Change the Mantle / Proof color split — "Proof" is always accent, never the other way
Use rounded corners on the icon — the geometric hard edge is intentional
